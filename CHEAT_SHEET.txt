üîê HOW XOR ENCRYPTION WORKS
================================================================================

In Python:
    original ^ key = encrypted
    encrypted ^ key = original

Example:
    >>> 5 ^ 3
    6
    >>> 6 ^ 3
    5

With bytes:
    original_byte = 234
    key_byte = 42
    encrypted_byte = original_byte ^ key_byte  # Encrypt
    decrypted_byte = encrypted_byte ^ key_byte  # Decrypt (back to 234!)

üîç COMMON ISSUES & SOLUTIONS
================================================================================

Problem: "Module not found: PIL"
Solution: Run: pip install --user Pillow

Problem: "File not found"
Solution: Make sure the file is in the same directory, or use full path

Problem: "Decryption failed"
Solution: Check password is exactly the same (case-sensitive!)

Problem: Image looks like garbage after decrypting
Solution: Wrong password was used, or file is corrupted

üìñ IMPORTANT CONCEPTS
================================================================================

Binary vs Text mode:
    'rb' = Read Binary (for images, encrypted files)
    'wb' = Write Binary
    'r'  = Read Text (for .txt files)
    'w'  = Write Text

Bytes vs Strings:
    "hello"           ‚Üí String (text)
    b"hello"          ‚Üí Bytes
    "hello".encode()  ‚Üí String to bytes
    b"hello".decode() ‚Üí Bytes to string

XOR operator:
    ^  ‚Üí XOR in Python
    Example: 5 ^ 3 = 6

Hash function:
    hashlib.sha256(password.encode()).digest()
    ‚Üí Converts any text to a fixed-size 32-byte key
