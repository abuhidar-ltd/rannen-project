===============================================
IMAGE ENCRYPTION TOOL - QUICK START GUIDE
===============================================

Hey! This is your super simple guide to get started.

===============================================
STEP 1: INSTALL
===============================================

Run this command:

    python3 setup.py

This installs everything you need.


===============================================
STEP 2: TRY THE DEMO
===============================================

See how it works with a demo:

    python3 demo.py

This creates a test image, encrypts it, and 
decrypts it. Watch and learn!


===============================================
STEP 3: ENCRYPT YOUR OWN IMAGE
===============================================

Run the main program:

    python3 main.py

Then:
1. Choose option 1 (Encrypt)
2. Enter your image filename (like: photo.jpg)
3. Enter a password
4. Done! You'll get a file called photo_encrypted.png


===============================================
STEP 4: DECRYPT IT BACK
===============================================

Run again:

    python3 main.py

Then:
1. Choose option 2 (Decrypt)
2. Enter the encrypted filename (photo_encrypted.png)
3. Enter the SAME password
4. Done! You'll get photo_decrypted.png


===============================================
WHAT'S HAPPENING?
===============================================

When you ENCRYPT:
1. Your image becomes bytes (numbers)
2. Password becomes an encryption key
3. XOR operation scrambles the bytes
4. Scrambled bytes saved to new file

When you DECRYPT:
1. Read the scrambled bytes
2. Password becomes same key
3. XOR again (it reverses!)
4. Image is recovered!
